*Wurzeln, Pronomen, Konnektoren, Zahlen — und das Prinzip der austauschbaren Wurzelschichten.*

## 1. Kernvokabular — Lateinisch/Germanische Wurzelschicht

Jede Wurzel ist so gewählt, dass ein Europäer sie intuitiv wiedererkennt.
Alle Wurzeln folgen der Phonologie: 5 Vokale (a, e, i, o, u), 12 Konsonanten (m, n, p, t, k, s, f, v, l, r, d, g), Silbenstruktur (K)V(K), keine Diphthonge.

Betonung: Immer auf der ersten Silbe der Wurzel (nach dem Marker).

- **-rug-** (Weg): Nicht direkt aus *Weg*, sondern lautlich eigenständig — aber die Assoziation "rauh/Richtung" funktioniert.
- Alle Wurzeln sind mit allen 8 Markern kombinierbar. Die Tabelle zeigt nur na-/va-/ka- als Beispiele.

## 2. Pronomen-System

ReLing-Pronomen sind Entitäten (Marker `na-`), aber als Kurzformen ohne Marker.

### Personalpronomen

| ReLing | Bedeutung | Herkunft/Assoziation |
|--------|-----------|---------------------|
| **me** | ich | lat. *me*, engl. *me* |
| **tu** | du | lat. *tu*, frz. *tu* |
| **se** | er/sie/es (geschlechtsneutral) | lat. *se* (reflexiv → 3. Person) |
| **nos** | wir | lat. *nos* |
| **vos** | ihr | lat. *vos* |
| **sen** | sie (Plural) | Erweiterung von *se* + Plural-Marker |

**Geschlechtsneutralität:** Es gibt kein grammatisches Geschlecht. `se` deckt alle 3. Personen ab. Bei Bedarf wird biologisches Geschlecht über Qualitäts-Modifikatoren ausgedrückt, nie über Pronomen.

### Demonstrativ- und Ortspronomen

| ReLing | Bedeutung | Assoziation |
|--------|-----------|-------------|
| **ki** | hier | kurz, nah klingend |
| **to** | dort | Distanz-Assoziation |
| **kes** | dies/diese(r) | ki + Entitäts-Bezug |
| **tos** | jenes/jene(r) | to + Entitäts-Bezug |

## 3. Konnektoren

Konnektoren verbinden Satzteile. Sie stehen als eigenständige Partikel — kein Marker nötig.

| ReLing | Bedeutung | Assoziation |
|--------|-----------|-------------|
| **e** | und | ital./span. *e* (und) |
| **o** | oder | ital./span. *o* (oder) |
| **sed** | aber | lat. *sed* (aber) |
| **kus** | weil | lat. *causa* (angepasst) |
| **si** | wenn/falls | lat./ital. *si* |
| **ke** | dass (Nebensatz-Einleiter) | frz. *que* |

## 4. Zahlensystem

Basis-10. Laute so gewählt, dass sie phonetisch klar trennbar sind.

| Zahl | ReLing | Assoziation |
|------|--------|-------------|
| 0 | **nul** | lat./dt. *null* |
| 1 | **un** | lat. *unus*, frz. *un* |
| 2 | **du** | lat. *duo* |
| 3 | **tri** | lat./griech. *tres/tri* |
| 4 | **kat** | (eigenständig, klar trennbar) |
| 5 | **pen** | griech. *penta* (angepasst) |
| 6 | **sis** | lat. *sex* (angepasst an Phonologie) |
| 7 | **sep** | lat. *septem* |
| 8 | **ok** | lat. *octo* (angepasst) |
| 9 | **nov** | lat. *novem* |
| 10 | **dek** | lat./griech. *decem/deka* |

**Bildungsregel:** 11 = dek-un, 12 = dek-du, 20 = du-dek, 100 = sen (lat. *centum*, angepasst), 1000 = mil (lat. *mille*).

**Achtung:** `nov` (9) und `-nov-` (neu) — Homonym? Nein: Zahlen stehen immer kontextfrei als Partikel. Die Wurzel `-nov-` tritt nur mit Marker auf (ka-nov, na-nov, va-nov). `nov` allein = 9. Keine Ambiguität.

## 5. ReLing lernen — Schritt für Schritt

### Schritt 1: Ein Wort bilden

Jedes ReLing-Wort besteht aus zwei Teilen: einem **Marker** vorne und einer **Wurzel** dahinter. Der Marker sagt, *was für eine Art Wort* es ist. Die Wurzel sagt, *worum es inhaltlich geht*.

Nehmen wir die Wurzel **-dom-** (Haus).

> **na-** bedeutet: *"Das Folgende ist ein Ding."*

**na-** + **-dom-** = **na-dom** → ein Ding, das mit Haus zu tun hat → **"das Haus"**

Noch ein Beispiel. Die Wurzel **-per-** (Mensch):

**na-** + **-per-** = **na-per** → ein Ding, das mit Mensch zu tun hat → **"die Person"**

**na-** macht aus jeder Wurzel ein Ding. Immer.

- na- + -lab- (Arbeit) = **na-lab** → **"die Arbeit"**
- na- + -lun- (Licht) = **na-lun** → **"das Licht"**
- na- + -akva- (Wasser) = **na-akva** → **"das Wasser"**

### Schritt 2: Ein anderer Marker, ein anderes Wort

Jetzt nehmen wir **dieselbe Wurzel -dom-** (Haus), aber einen anderen Marker:

> **ka-** bedeutet: *"Das Folgende ist eine Eigenschaft."*

**ka-** + **-dom-** = **ka-dom** → eine Eigenschaft, die mit Haus zu tun hat → **"häuslich"**

Und noch einer:

> **va-** bedeutet: *"Das Folgende ist eine Handlung."*

**va-** + **-dom-** = **va-dom** → eine Handlung, die mit Haus zu tun hat → **"behausen"**

Drei Wörter, eine Wurzel:

| Wort | Marker | Bedeutung |
|------|--------|-----------|
| **na-dom** | na- (*Ding*) | das Haus |
| **ka-dom** | ka- (*Eigenschaft*) | häuslich |
| **va-dom** | va- (*Handlung*) | behausen |

### Schritt 3: Das Prinzip an einer zweiten Wurzel

Damit es sitzt — dasselbe mit **-lab-** (Arbeit):

- na- (*Ding*) + -lab- = **na-lab** → **"die Arbeit"** (ein Ding)
- ka- (*Eigenschaft*) + -lab- = **ka-lab** → **"arbeitend"** (eine Eigenschaft)
- va- (*Handlung*) + -lab- = **va-lab** → **"arbeiten"** (eine Handlung)

Und mit **-bon-** (gut):

- na- (*Ding*) + -bon- = **na-bon** → **"das Gute"** (ein Ding)
- ka- (*Eigenschaft*) + -bon- = **ka-bon** → **"gut"** (eine Eigenschaft)
- va- (*Handlung*) + -bon- = **va-bon** → **"verbessern"** (eine Handlung)

Das Muster ist immer dasselbe:
- **na-** macht ein Ding daraus
- **ka-** macht eine Eigenschaft daraus
- **va-** macht eine Handlung daraus

### Schritt 4: Der vierte Marker — Verbindungen

> **ra-** bedeutet: *"Das Folgende ist eine Verbindung/Beziehung."*

- ra- (*Verbindung*) + -lab- (Arbeit) = **ra-lab** → **"durch Arbeit"**
- ra- (*Verbindung*) + -dom- (Haus) = **ra-dom** → **"vom Haus her"**
- ra- (*Verbindung*) + -per- (Mensch) = **ra-per** → **"von der Person"**

**ra-** sagt: Die Wurzel beschreibt nicht ein Ding oder eine Handlung, sondern eine Beziehung zwischen Dingen.

### Schritt 5: Erste Sätze

Jetzt haben wir genug Bausteine für einfache Sätze.

**Satz 1:** *"Die Person arbeitet."*
```
na-per    va-lab
na-per   va-lab
Ding:    Handlung:
Person   arbeiten
```

**Satz 2:** *"Die Person gibt ein Haus."*
```
na-per    va-don    na-dom
na-per   va-don   na-dom
Ding:    Handlung: Ding:
Person   geben    Haus
```

**Satz 3:** *"Die Person gibt ein gutes Haus."*
```
na-per    va-don    na-dom    ka-bon
na-per   va-don   na-dom   ka-bon
Ding:    Handlung: Ding:    Eigenschaft:
Person   geben    Haus     gut
```

**Satz 4:** *"Die arbeitende Person gibt ein großes Haus."*
```
na-per    ka-lab      va-don    na-dom    ka-mag
na-per   ka-lab     va-don   na-dom   ka-mag
Ding:    Eigensch.: Handlung: Ding:    Eigenschaft:
Person   arbeitend  geben    Haus     groß
```

### Schritt 6: Die Wurzel wandert durch den Satz

Jetzt zeigen wir den Kern von ReLing: **-lab-** (Arbeit) taucht in jedem Satz auf — aber jedes Mal mit einem anderen Marker in einer anderen Rolle.

**-lab- als Handlung:** *"Die Person arbeitet."*
```
na-per    va-lab
na-per   va-lab
Ding:    Handlung:
Person   arbeiten
```

**-lab- als Ding:** *"Die Arbeit ist gut."*
```
na-lab    ka-bon
na-lab   ka-bon
Ding:    Eigenschaft:
Arbeit   gut
```

**-lab- als Eigenschaft:** *"Die arbeitende Person gibt ein Haus."*
```
na-per    ka-lab      va-don    na-dom
na-per   ka-lab     va-don   na-dom
Ding:    Eigensch.: Handlung: Ding:
Person   arbeitend  geben    Haus
```

**-lab- als Verbindung:** *"Die Person gibt ein Haus durch Arbeit."*
```
na-per    va-don    na-dom    ra-lab
na-per   va-don   na-dom   ra-lab
Ding:    Handlung: Ding:    Verbindung:
Person   geben    Haus     durch Arbeit
```

Selbe Wurzel. Der Marker entscheidet die Rolle.

### Schritt 7: Der Rahmen — Wie ist es gemeint?

Bisher haben wir nur den **Kern** gebaut (wer tut was). Jetzt kommt der **Rahmen** davor. Er sagt: *Wie ist das Folgende gemeint?*

Dafür gibt es drei weitere Marker:

> **sa-** bedeutet: *"Das Folgende ist eine Absicht."* (Aussage? Frage? Bitte?)
> **ma-** bedeutet: *"Das Folgende ist eine Einschätzung."* (Sicher? Vielleicht?)
> **ta-** bedeutet: *"Das Folgende ist eine Zeitangabe."* (Jetzt? Morgen? Gestern?)

Drei neue Wörter:

- sa- (*Absicht*) + -vel- (mitteilen) = **sa-vel** → **"ich sage hiermit"** (Aussage)
- ma- (*Einschätzung*) + -dul- (unsicher) = **ma-dul** → **"vermutlich"**
- ta- (*Zeit*) + -nes- (nächster) = **ta-nes** → **"morgen"**

Der Satz: *"Vermutlich gibt die Person morgen ein gutes Haus."*
```
sa-vel     ma-dul        ta-nes     na-per    va-don    na-dom    ka-bon
sa-vel     ma-dul        ta-nes     na-per   va-don   na-dom   ka-bon
Absicht:   Einschätzung: Zeit:      Ding:    Handlung: Ding:    Eigensch.:
Aussage    vermutlich    morgen     Person   geben    Haus     gut
╰──────────Rahmen──────────╯        ╰──────Kern───────╯        ╰Färbung╯
```

### Schritt 8: Affekt — Was fühlt man dabei?

Der letzte Marker:

> **fa-** bedeutet: *"Das Folgende ist ein Gefühl."*

Besonderheit: Der Vokal im Marker ändert sich je nach Gefühl:

| Marker | Bedeutung | Beispiel |
|--------|-----------|----------|
| **fa-** | neutral, sachlich | fa-gol = Freude, sachlich festgestellt |
| **fe-** | leicht positiv | fe-gol = angenehme Freude |
| **fi-** | stark positiv | fi-gol = Begeisterung |
| **fo-** | leicht negativ | fo-gol = getrübte Freude |
| **fu-** | stark negativ | fu-gol = bittere Freude |

Der vollständige Satz: *"Vermutlich gibt die Person morgen freudig ein gutes Haus."*
```
sa-vel     ma-dul        ta-nes     na-per    va-don    na-dom    ka-bon       fe-gol
Absicht:   Einschätzung: Zeit:      Ding:    Handlung: Ding:    Eigensch.:  Gefühl:
Aussage    vermutlich    morgen     Person   geben    Haus     gut         Freude(+)
╰──────────Rahmen──────────╯        ╰──────Kern───────╯        ╰────Färbung────╯
```

### Schritt 9: Bausteine tauschen — ein Wort ändern, Satz ändert sich

Wir nehmen den fertigen Satz und tauschen jedes Mal **ein** Wort.

**Original:** *"Vermutlich gibt die Person morgen freudig ein gutes Haus."*
```
sa-vel       ma-dul          ta-nes    na-per     va-don      na-dom     ka-bon        fe-gol
Absicht:     Einschätzung:   Zeit:     Ding:     Handlung:  Ding:     Eigensch.:   Gefühl:
Aussage      vermutlich      morgen    Person    geben      Haus      gut          Freude(+)
```

**Absicht ändern** — sa-vel (Aussage) → sa-rik (*Absicht:* Frage):
*"Gibt die Person vermutlich morgen freudig ein gutes Haus?"*
```
sa-rik       ma-dul          ta-nes    na-per     va-don      na-dom     ka-bon        fe-gol
Absicht:     Einschätzung:   Zeit:     Ding:     Handlung:  Ding:     Eigensch.:   Gefühl:
Frage        vermutlich      morgen    Person    geben      Haus      gut          Freude(+)
```

**Einschätzung ändern** — ma-dul (vermutlich) → ma-for (*Einschätzung:* sicher):
*"Sicher gibt die Person morgen freudig ein gutes Haus."*
```
sa-vel       ma-for          ta-nes    na-per     va-don      na-dom     ka-bon        fe-gol
Absicht:     Einschätzung:   Zeit:     Ding:     Handlung:  Ding:     Eigensch.:   Gefühl:
Aussage      sicher          morgen    Person    geben      Haus      gut          Freude(+)
```

**Zeit ändern** — ta-nes (morgen) → ta-ki (*Zeit:* jetzt):
*"Vermutlich gibt die Person jetzt freudig ein gutes Haus."*
```
sa-vel       ma-dul          ta-ki     na-per     va-don      na-dom     ka-bon        fe-gol
Absicht:     Einschätzung:   Zeit:     Ding:     Handlung:  Ding:     Eigensch.:   Gefühl:
Aussage      vermutlich      jetzt     Person    geben      Haus      gut          Freude(+)
```

**Handlung ändern** — va-don (*Handlung:* geben) → va-vid (*Handlung:* sehen):
*"Vermutlich sieht die Person morgen freudig ein gutes Haus."*
```
sa-vel       ma-dul          ta-nes    na-per     va-vid      na-dom     ka-bon        fe-gol
Absicht:     Einschätzung:   Zeit:     Ding:     Handlung:  Ding:     Eigensch.:   Gefühl:
Aussage      vermutlich      morgen    Person    sehen      Haus      gut          Freude(+)
```

**Gefühl ändern** — fe-gol (*Gefühl:* Freude+) → fo-gol (*Gefühl:* Freude–):
*"Vermutlich gibt die Person morgen bedrückt ein gutes Haus."*
```
sa-vel       ma-dul          ta-nes    na-per     va-don      na-dom     ka-bon        fo-gol
Absicht:     Einschätzung:   Zeit:     Ding:     Handlung:  Ding:     Eigensch.:   Gefühl:
Aussage      vermutlich      morgen    Person    geben      Haus      gut          Freude(–)
```

Jeder Slot ist unabhängig. Ein Wort tauschen, der Rest bleibt.

## 6. Beispielsatz — Lateinisch/Germanische Wurzeln

### Deutsch:
> "Hallo, Fremder, bist du neu hier? Es gibt viel zu sehen."

### ReLing:

```
fe-gol,  na-gast,  sa-rik  tu  ka-nov  ki?   va-don  ka-mul  va-vid.
```

### Wort-für-Wort-Analyse:

| ReLing | Marker | Wurzel/Wort | Bedeutung |
|--------|--------|-------------|-----------|
| **fe-gol** | fa→fe (positiv-leicht) | -gol- (Freude) | Freudiger Gruß (= Hallo) |
| **na-gast** | na- (Entität) | -gast- (Fremd) | Fremder, Gast |
| **sa-rik** | sa- (Intention) | -rik- (Frage) | Intention: Frage |
| **tu** | — (Pronomen) | tu | du |
| **ka-nov** | ka- (Qualität) | -nov- (Neu) | neu |
| **ki** | — (Ortspronomen) | ki | hier |
| **va-don** | va- (Aktion) | -don- (Geben/Existieren) | es gibt / existiert |
| **ka-mul** | ka- (Qualität) | -mul- (Viel) | viel |
| **va-vid** | va- (Aktion) | -vid- (Sehen) | zu sehen |

### Satzstruktur:

```
[Affekt:Gruß] [Entität:Anrede] — [Intention:Frage] [Entität:du] [Qualität:neu] [Ort:hier] — [Aktion:existieren] [Qualität:viel] [Aktion:sehen]
```

Rahmen → Kern → Kern. Zwei Aussagen, durch Satzgrenze (`.` bzw. `?`) getrennt.

**Anmerkung:** `sa-rik` nutzt die neue Wurzel `-rik-` (von germ. *richten*, engl. *request* — Frage/Anfrage). Diese Wurzel gehört zum Intention-Feld und wird hauptsächlich mit `sa-` verwendet.

## 7. Derselbe Satz — Chinesisch-inspirierte Wurzelschicht

Dieselbe Grammatik, dieselben Marker — nur andere Wurzeln, abgeleitet aus dem Mandarin-Chinesischen und angepasst an ReLing-Phonologie.

### Chinesische Wurzeln (angepasst)

| Bedeutungsfeld | Mandarin-Quelle | ReLing-Adaptation | Erläuterung |
|----------------|----------------|-------------------|-------------|
| Freude/Gruß | 喜 xǐ (Freude) | **-si-** | xǐ → si (x→s, ǐ→i) |
| Fremd/Gast | 客 kè (Gast) | **-ke-** | kè → ke (direktübernahme) |
| Frage | 问 wèn (fragen) | **-ven-** | wèn → ven (w→v) |
| Neu | 新 xīn (neu) | **-sin-** | xīn → sin (x→s) |
| Hier | 这里 zhèlǐ | **-li-** | lǐ → li |
| Geben/Existieren | 有 yǒu (existieren) | **-u-** | yǒu → u |
| Viel | 多 duō (viel) | **-do-** | duō → do (kein Diphthong) |
| Sehen | 看 kàn (sehen) | **-kan-** | kàn → kan (direkt) |

### Der Satz:

```
fe-si,   na-ke,   sa-ven   tu   ka-sin   li?   va-u   ka-do   va-kan.
```

### Wort-für-Wort:

| ReLing (CN) | Marker | Wurzel | Bedeutung |
|-------------|--------|--------|-----------|
| **fe-si** | fa→fe (positiv-leicht) | -si- (Freude) | Freudiger Gruß |
| **na-ke** | na- (Entität) | -ke- (Gast) | Fremder, Gast |
| **sa-ven** | sa- (Intention) | -ven- (Frage) | Intention: Frage |
| **tu** | — (Pronomen) | tu | du |
| **ka-sin** | ka- (Qualität) | -sin- (Neu) | neu |
| **li** | — (Ortspronomen) | li | hier |
| **va-u** | va- (Aktion) | -u- (Existieren) | es gibt |
| **ka-do** | ka- (Qualität) | -do- (Viel) | viel |
| **va-kan** | va- (Aktion) | -kan- (Sehen) | zu sehen |

### Vergleich:

```
Europäisch:  fe-gol,  na-gast,  sa-rik  tu  ka-nov  ki?   va-don  ka-mul  va-vid.
Chinesisch:  fe-si,   na-ke,    sa-ven  tu  ka-sin  li?   va-u    ka-do   va-kan.
```

**Identische Struktur.** Nur die Wurzeln und Ortspronomen ändern sich. Marker, Satzstellung, Modifikatoren — alles bleibt gleich. Ein Chinese erkennt `-kan-` (看) und `-sin-` (新) sofort. Ein Europäer erkennt `-vid-` und `-nov-`. Beide verstehen denselben Satz.

## 8. Wurzelschicht-System — Das zentrale Designprinzip

### Architektur

```
┌─────────────────────────────────────┐
│  GRAMMATIK (statisch, universal)    │
│  Marker, Modifikatoren, Satzstruktur│
│  Konnektoren, Pronomen, Zahlen      │
│  → Identisch für ALLE Nutzer        │
├─────────────────────────────────────┤
│  WURZELSCHICHT (regional, wählbar)  │
│  ┌───────────┐ ┌──────────┐        │
│  │ Europäisch│ │ Sinisch  │  ...   │
│  │ -vid-     │ │ -kan-    │        │
│  │ -nov-     │ │ -sin-    │        │
│  │ -gast-    │ │ -ke-     │        │
│  └───────────┘ └──────────┘        │
└─────────────────────────────────────┘
```

### Regeln

1. **Grammatik = statisch, universal.**
   Die 8 Marker (na-, va-, ka-, ra-, ma-, ta-, sa-, fa-), die Modifikatoren (-in, -es, -ul, -om, -an), die Satzstruktur (Rahmen → Kern → Färbung) und die Konnektoren sind für alle Nutzer identisch. Sie werden nie ausgetauscht.

2. **Wurzeln = regional austauschbar.**
   Jedes Bedeutungsfeld hat mindestens eine Wurzel pro Sprachfamilie. Die Wurzel trägt die Semantik, der Marker die Funktion. Verschiedene Wurzelschichten drücken dasselbe aus — nur der Wiedererkennungseffekt richtet sich nach der Muttersprache des Nutzers.

3. **Trans wählt die Wurzelschicht passend zum Nutzer.**
   Ein deutschsprachiger Nutzer bekommt die europäische Schicht. Ein chinesischer die sinische. Trans übersetzt nicht nur *in* die Zielsprache, sondern wählt auch die ReLing-Wurzeln, die dem Nutzer am nächsten liegen — falls der Nutzer ReLing direkt lesen will.

4. **Mehrere Schichten können koexistieren.**
   Es gibt kein "richtiges" ReLing-Wort für "sehen". Es gibt `-vid-` (europäisch) und `-kan-` (sinisch) und potentiell `-mir-` (arabisch, von نظر naẓara, angepasst) und weitere. Alle sind gleichberechtigt. Die Grammatik entscheidet über Bedeutung, nicht die Wurzel.

### Geplante Wurzelschichten

| Schicht | Sprachfamilien-Basis | Status |
|---------|---------------------|--------|
| Europäisch | Latein, Germanisch, Englisch, Romanisch | v0.1 (diese Datei) |
| Sinisch | Mandarin, Kantonesisch | Prototyp (diese Datei) |
| Arabisch | Arabisch, Persisch | geplant |
| Indisch | Sanskrit, Hindi | geplant |
| Afrikanisch | Swahili, Bantu-Sprachen | geplant |
| Austronesisch | Indonesisch, Tagalog | geplant |

### Warum das funktioniert

ReLing trennt Struktur von Klang. Die Struktur ist die Sprache. Der Klang ist die Brücke zum Nutzer. Damit wird ReLing nicht "eine weitere europäische Kunstsprache" (wie Esperanto), sondern ein System, das sich jeder Sprachgemeinschaft anpasst — ohne die Grammatik zu verändern.
